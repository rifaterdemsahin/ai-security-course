Quick Start Guide

1. Prerequisites:

  • Python 3.7+ installed
  • Basic command line knowledge
  • Text editor or IDE

2. Installation:

pip install numpy matplotlib tensorflow scikit-learn

3. Navigate to demos:

cd "5_Symbols/Lessons/Lesson 1/"

4. Run launcher:

python demo_launcher.py
Recommended Learning Path
  1. Start with Evasion Demo - Most intuitive introduction to AI vulnerabilities
  2. Explore Data Poisoning - Understand training-time threats
  3. Try Model Extraction - See intellectual property risks
  4. Use Assessment Tool - Learn systematic security evaluation
